# News Bot (Crunchyroll + YouTube) -> Telegram + Website API

This bot:
- Pulls the **latest Crunchyroll RSS** entry (ar-SA)
- If new:
  - Adds logo to the image
  - Sends the post to **Telegram**
  - Uploads the article to your website API (`new-article.php`) with:
    - title
    - description
    - time
    - image (WebP)
- Pulls the **latest YouTube** video and sends to Telegram if new
